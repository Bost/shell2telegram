shell2telegram
==============

Create Telegram bot from command-line

Usage:

Get token from [BotFather bot](https://telegram.me/BotFather), and set TB_TOKEN var in shell

    export TB_TOKEN=********
    shell2telegram /cmd 'shell command' /cmd2 'shell command2'
